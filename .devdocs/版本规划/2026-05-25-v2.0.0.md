## 三、向后兼容性评估

- [x] **API** — Breaking: `MemoryScope` → `MemoryScopeV2`，提供 `adaptScopeV1toV2` / `adaptScopeV2toV1` 适配器
- [x] **配置** — 无 breaking changes（scope 是运行时行为，非配置项）
- [x] **数据库** — Schema v2 自动迁移，幂等，旧数据 `scope_session` → `scope_chat` 自动映射
- [x] **旧 scope 字段保留并标记 @deprecated** — BmNode 同时保留 v1 和 v2 字段
- [x] **迁移指南 S-4** — 涵盖 scope 变更、数据库迁移、API 变更、回滚方案
