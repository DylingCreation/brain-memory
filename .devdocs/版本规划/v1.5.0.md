## 三、问题清单与修复方案

### P1 — Lite 模式（#12）

**问题描述**：
使用 qwen3.5:9b（Ollama）等本地小模型时，reflection 和 fusion 的 prompt 包含大量结构化上下文，生成速度不够快，60s 超时不够。3 个 LLM 集成测试全部超时。

**影响**：本地模型开发者无法完整运行测试，日常 Agent 场景中 reflection/fusion 延迟过高。

**修复方案**：

新增 `mode: "lite"` 配置项：

```typescript
interface BmConfig {
  // 现有字段...
  
  /** 运行模式：full=全部功能(默认), lite=跳过 reflection/fusion */
  mode?: "full" | "lite";
}
```

Lite 模式行为：
- ✅ 保留：启发式提取 + 向量召回 + 工作记忆 + 增量图维护
- ❌ 跳过：LLM 知识提取、session reflection、知识融合

### P2 — 作者身份（#13）
