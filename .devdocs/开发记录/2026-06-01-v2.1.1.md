## 四、关键修复说明

### F-2: plugin-core config 根因

`BrainMemoryPluginCore` 构造函数中 `deepMergePluginConfig` 的 defaults 参数仅包含 4 个 plugin 开关字段（`enabled/injectMemories/extractMemories/autoMaintain`），缺少 `BmConfig` 的 30+ 个必需字段（`storage/engine/decay/reflection/fusion/...`）。导致 `init()` 创建 `ContextEngine` 时传入不完整配置，`onSessionEnd` 链路中 `runDecayArchiving` 访问 `cfg.decay.enabled` 时 `cfg.decay` 为 `undefined`。

**修复**：将 base config 从 sparse plugin defaults 改为 `DEFAULT_CONFIG`，plugin defaults 作为顶层覆盖。
