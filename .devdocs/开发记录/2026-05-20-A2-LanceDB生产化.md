## 1. 开发内容

v1.3.0 POC → v1.6.0 L2 基础可用级。

### 改进

| 领域 | 改进前 | 改进后 |
|------|--------|--------|
| **错误处理** | `initialize()` 无 try/catch，失败时静默 | try/catch + `_initError` 诊断字段 + 清晰错误消息 |
| **向量持久化** | `saveVector()` 仅写内存 | 写内存（立即生效）+ LanceDB 表 fire-and-forget（持久化） |
| **向量搜索** | 纯内存 cosine | 保持内存（满足 IStorageAdapter 同步接口），LanceDB 作为持久层 |
| **LanceDB 表** | 开连接但从未建表/读写 | 自动创建 `bm_vectors` 表，`saveVector` 写入 |
| **close()** | 无错误处理 | try/catch 包装，best-effort 关闭 |
| **测试** | 4 个冒烟测试 | 15 个测试（生命周期/CRUD/向量/边/dirty marks） |

### 架构定位

```
┌─ IStorageAdapter ───────────────────────────────┐
│                                                  │
│  SQLiteStorageAdapter  ← 生产主力（全部能力）     │
│  LanceDBStorageAdapter ← 向量后端备选（v1.6.0）  │
│                          向量操作 ✅              │
│                          Node/Edge 内存 stub      │
│                          社区/消息 stub          │
└──────────────────────────────────────────────────┘
```
