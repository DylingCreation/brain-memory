## 四、测试失败修复记录

### 4.1 decay-presets — 衰减曲线跨层比较断言错误

**根因**：decay 公式 `exp(-pow(days/λ, beta))` 中，beta 参数对曲线形态的影响是非线性的。在 day 1 极短期，`betaCore=0.8` 比 `betaWorking=1.0` 衰减更快，与直觉相反。

**修复**：删除跨层比较断言，改为验证每个 tier 自身单调递减 + day 365 到达 floor。

### 4.2 reranker — embedFn 异常处理

**根因**：源码 catch 块静默跳过（不 set 分数），测试预期设 0 分，不一致。

**修复**：断言改为 `has('n1') === false`。

### 4.3 reranker — API 路径单节点短路

**根因**：`rerank()` 方法 `nodes.length <= 1` 时直接返回（短路），pinecone 和 data 字段测试只传了 1 个节点。

**修复**：测试增加第 2 个节点。

### 4.4 buildEmbeddingText — sentence boundary 位置

**根因**：句子边界搜索范围是 `cutPoint * 0.5` 到 `cutPoint`（即 600-1200），测试把句号放在 600 位置，循环 `i > 600` 不检查。

**修复**：把标点符号放在 900 位置（> 840）。

### 4.5 buildEmbeddingText — paragraph break 位置

**根因**：paragraphBreak 需要 > 720 才进入该分支，测试 `\n\n` 在 626 位置。

**修复**：调整 `\n\n` 到 800 位置，加 header 后 > 720。

### 4.6 recaller-extended — describe 块缺少闭合括号

**根因**：编辑时漏掉了 `});`。

**修复**：补充闭合。
