## 1. 开发内容

### 通用增强（Full + Small 共用）

| 模块 | 改进前 | 改进后 |
|------|--------|--------|
| `reasoning/engine.ts` | 内联 `JSON.parse(s)`，无 tolerant | 使用 `extractJsonTolerant()` — 引号修正/尾随逗号/代码块剥离/括号补全 |
| `fusion/analyzer.ts` | 同上，内联裸 JSON.parse | 同上 |
| `reflection/extractor.ts` | 使用 `extractJson()`（标准提取，无修复） | 使用 `extractJsonTolerant()`（自动修复） |
| `extractor/extract.ts` | 已使用 tolerant ✅ | 无变更 |

### Small 专用（仅在 `mode: "small"` 时激活，为 B-1 铺路）

| 新增 | 说明 |
|------|------|
| `smallJsonRepair()` | 在通用 tolerant 基础上增加 Small 模型特有修复：字符级包装、bare text→conclusion |
| `safeJsonParse()` | 安全解析，失败返回 fallback 不抛异常 |
| `SMALL_DEFAULTS` | 节点默认值（type/category/source），Small 模型输出缺失字段时填充 |
