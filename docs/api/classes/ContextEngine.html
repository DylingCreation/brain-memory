    Class ContextEngine

    统一上下文引擎 — brain-memory 主入口。

    整合提取、召回、融合、反思、推理和工作记忆。 通过 IStorageAdapter 实现存储抽象，Recaller 处理双路径召回。

    Example

    const engine = new ContextEngine({ dbPath: ":memory:" });
    await engine.processTurn({ ... });
    const result = await engine.recall("query");
